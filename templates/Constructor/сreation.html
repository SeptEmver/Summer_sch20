Привет! Прежде чем приступить к созданию своей уникальной игры, просим ознакомиться с краткой инструкцией:

  1. Для того, чтобы приступить к созданию уровней, выберите нужный уровень из списка слева.
  2. Далее необходимо задать размеры игрового поля. Ширина и высота игрового поля могут быть минимум 4 блока и максимум 16.
  3. Ячейки игрового поля можно редактировать по клику на них. Вы можете выбрать сразу несколько ячеек для редактирования. Кликнув по одной ячейке дважды, вы можете отменить ее выделение. Сбросить выделение ячеек также можно, нажав клавишу Escape.
  4. После того, как вы выберите ячейки, справа от игрового поля появится окно, в котором можно выбрать тип для всех выделенных ячеек. Для этого нужно выбрать желаемое свойство и нажать кнопку "Применить". Важно! На игровом поле нельзя расположить более одного игрока.
  5. Данная игра имеет текстовое представление. Для каждого типа ячейки мы задали вид-символ по умолчанию, но вы можете его изменить. Для этого необходимо обратиться к полю выбора символа и вписать желаемый символ для отдельных объектов. Вы можете ввести любой символ, который есть на вашей клавиатуре (кроме пробела) и применить изменения, нажав кнопку "Применить".
